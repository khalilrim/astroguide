# AstroGuide - Encyclopédie de l’Astronomie

## Thème
Ce projet est une encyclopédie web dédiée à l’astronomie.  
Il permet de découvrir les planètes, les étoiles, les galaxies et les phénomènes cosmiques à travers différentes pages.
